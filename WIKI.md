## Features at a Glance

| Feature | Description |
|---------|-------------|
| **Multi-format Support** | Parse Maven, Gradle, npm, and PIP dependencies |
| **Real-time Scanning** | Automatic vulnerability detection as you work |
| **OSV Integration** | Query the Open Source Vulnerabilities database |
| **GitHub Advisories** | Secondary source for security advisories |
| **SARIF Export** | Export results for CI/CD integration |
| **Auto-Fix** | Quick fixes to update vulnerable dependencies |
| **License Scanning** | Check licenses against allowed lists |
