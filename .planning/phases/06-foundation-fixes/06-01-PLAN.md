<objective>
Fix five broken or non-functional foundation features identified during competitive research against Snyk, Mend.io, and Qodana. All fixes must reach production quality matching Mend.io Package Checker (bundled free).

**Features to Fix (in priority order):**

1. **CVSS Severity Parsing** — `OsVApiService.kt` at line ~222 hardcodes `severity = OsVSeverity.MEDIUM`. The OSV API returns `severity: [{"type": "CVSS_V3", "score": "9.8"}]` which is ignored. Every vulnerability shows as MEDIUM regardless of actual severity.

2. **License Scanning** — `LicenseParser.kt` at line ~14 guesses licenses from package names (!) instead of querying registries. Returns UNKNOWN for 90%+ packages. The UI exists and `LicenseConflictDetector` has a full compatibility matrix, but has no real data.

3. **Synchronous Inspections** — `OsVInspection.kt` at line ~57 runs `scanFileForDependencies()` synchronously in `visitFile()`. Every file open triggers blocking network calls, freezing the IDE UI.

4. **String-Based Auto-Fix** — `OsVQuickFix.kt` at line ~90 reads the entire file as a String, does line-by-line regex replacement, then writes back. No PSI awareness — breaks XML/JSON structure if version string appears elsewhere. No undo support.

5. **Batch API Queries** — OSV POST `/v1/querybatch` consistently returns `{}`. Current code falls back to sequential individual queries which is order-of-magnitude slower.

**Why:** These foundation bugs prevent the plugin from being competitive. Snyk, Mend, and Qodana all have these basics working correctly.
</objective>

      ```kotlin
      class OsVInspection : LocalInspectionTool() {
          private val fileCache = ConcurrentHashMap<String, List<VulnerabilityResult>>()
          private val debounceTimers = ConcurrentHashMap<String, Timer>()

          override fun buildVisitor(holder: ProblemsHolder, isOnTheFly: Boolean): PsiElementVisitor {
              return object : PsiElementVisitor() {
                  override fun visitFile(file: PsiFile) {
                      val filePath = file.virtualFile?.path ?: return
                      val cached = fileCache[filePath]
                      if (cached != null) {
                          cached.forEach { reportCachedResult(holder, it) }
                          return
                      }

                      // Debounce: cancel existing timer, start new one
                      debounceTimers[filePath]?.cancel()
                      val timer = Timer().apply {
                          schedule(object : TimerTask() {
                              override fun run() {
                                  scanFileAsync(file, holder)
                              }
                          }, 500)
                      }
                      debounceTimers[filePath] = timer
                  }
              }
          }

          private fun scanFileAsync(file: PsiFile, holder: ProblemsHolder) {
              ProgressManager.getInstance().run(
                  object : Backgroundable(file.project, "Scanning dependencies...") {
                      override fun run(indicator: ProgressIndicator) {
                          val results = scanFile(file)
                          ApplicationManager.getApplication().invokeLater {
                              results.forEach { reportCachedResult(holder, it) }
                          }
                      }
                  }
              )
          }
      }
      ```

      ```kotlin
      fun batchQueryVulnerabilitiesParallel(deps: List<Dependency>): Map<Dependency, List<Vulnerability>> {
          val results = ConcurrentHashMap<Dependency, List<Vulnerability>>()
          val semaphore = Semaphore(10) // Max 10 concurrent
          val latch = CountDownLatch(deps.size)

          for (dep in deps) {
              semaphore.acquire()
              val cacheKey = "${dep.name}:${dep.ecosystem}:${dep.version}"
              cacheManager.getCachedVulnerabilities(cacheKey)?.let {
                  results[dep] = it
                  semaphore.release()
                  latch.countDown()
              } ?: run {
                  val request = buildQueryRequest(dep.name, dep.ecosystem, dep.version)
                  val call = httpClient.newCall(Request.Builder()
                      .url(osvApiUrl)
                      .post(RequestBody.create("application/json".toMediaType(), request))
                      .build())

                  call.enqueue(object : Callback {
                      override fun onFailure(call: Call, e: IOException) {
                          results[dep] = emptyList()
                          semaphore.release()
                          latch.countDown()
                      }
                      override fun onResponse(call: Call, response: Response) {
                          try {
                              val body = response.body?.string()
                              if (response.isSuccessful && body != null) {
                                  results[dep] = parseVulnerabilities(body)
                              } else {
                                  results[dep] = emptyList()
                              }
                          } catch (e: Exception) {
                              results[dep] = emptyList()
                          }
                          semaphore.release()
                          latch.countDown()
                      }
                  })
              }
          }

          latch.await(60, TimeUnit.SECONDS)
          return results
      }
      ```
